# Sistema de Gestión de Guardias y Presencia

Aplicación desarrollada en Python con Flask que permite:

- Registrar la presencia del profesorado
- Detectar ausencias automáticamente
- Gestionar horarios
- Calcular guardias

El sistema se basa en una arquitectura modular separando:
- Capa de datos (SQLite)
- Lógica
- Interfaz web
